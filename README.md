### ⏱️ Vantagens da Automação
- Antes: Perda de SLA - E-mails perdidos ou apagados
- Depois: Captura dos e-mails de forma automatica com automação Power Automate
- Este mesmo modelo pode ser reaproveitado para muitos outros objetivos com apenas alguns ajustes, exemplo :
- (Pedidos de Reembolso)
- (Aviso de Férias)
- Fluxos de Aprovação de Compras

---

### ⏱️ Vantagens da Automação
- Antes: 6h/dia de trabalho manual
- Depois: 15 minutos com automação Power Automate
- Envio de notificações direto no Teams do usuário durante o processo
